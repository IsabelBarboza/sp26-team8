## Description
> CleanSweep is a web platform that helps households and small offices easily find reliable cleaning service providers, including companies and independent professionals.

The main goal of CleanSweep is to make it simple for customers to book trusted cleaning services, while giving providers a single place to manage their offerings, schedules, and customer feedback.

## App Functions

### Customer (Isabel Barboza):
1. Create/modify customer profile - Customers can enter their personal information, like name, address, phone number, and cleaning choices through a web form.
2. View available services - Customers can search services by type, ratings, location and availability.
3. Subscribe to available services - Customers can schedule single or regular cleaning appointments by choosing a provider and time.
4. Write reviews for subscribed services - Customers can leave ratings and feedback after completed services.

### Provider (the user with the provider role):
1. Create/modify/remove provider profile - providers can enter and edit their profesional info through a web form 
2. Create services - providers can set an availabuility during which they can be subscrived to 
3. View customer statistics - providers can see the bio and history of customers that have booked them
4. Reply to reviews - providers can through their bio view and respond to cusstomers reviews 

### SysAdmin (the user with the admin role if applicable): 
1. Manage user access - Empty
2. Moderate services - Empty
3. Moderate reviews - Empty
4. View usage statistics - Empty
